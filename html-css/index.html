Interfaz web, para el proyecto de Voro y Jaume para Interfaces Web

Este proyecto muestra una página web responsive creada solo con HTML y CSS, cuidando el diseño, la accesibilidad y la usabilidad.Intentando cumplir todos requisitos pedidos en el dossier que se nos ha propuesto via la actividad

Ver más detalles

Características principales

Diseño responsive

La página se adapta a móviles, tablet y escritorio usando flexbox, grid y media queries.

Menú accesible

El menú superior se convierte en un menú hamburguesa en pantallas pequeñas para facilitar la navegación.

Buena legibilidad

Se utilizan colores con contraste suficiente, tamaños de texto cómodos y jerarquía clara de títulos.
